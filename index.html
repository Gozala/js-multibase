Intro

Installable via npm install --save multibase, it can also be used directly in the browser.

Quick Example

'use strict'

const multibase = require('multibase')

const encodedBuf = multibase.encode('base58btc', Buffer.from('hey, how is it going'))

const decodedBuf = multibase.decode(encodedBuf)
console.log(decodedBuf.toString()) // eslint-disable-line no-console
// => hey, how it going

There are more available, so take a look at the docs below for a full list. This documentation aims to be comprehensive, so if you feel anything is missing please create a GitHub issue for it.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save multibase

You can then require() multibase as normal:

const multibase = require('multibase')

In the Browser

multibase should work in any ECMAScript 2018 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of multibase, including index.js and index.min.js, are included in the /dist folder. multibase can also be found on unpkg.com under

  • https://unpkg.com/multibase/dist/index.min.js
  • https://unpkg.com/multibase/dist/index.js
